# Troubleshoot
## Resetting the sensor

In Nova, you can click the three dots in the top-right corner to reset the application or access the file explorer for **deleting files**.  

![image](../images/reset.png)  

Click **_Application defaults_** to reset all the settings and jobs that were created.  

To delete all the images created during an educational training session, head to Nova's **_file explorer_**, right-click the _/public/Nova_ folder and click _Delete_.
