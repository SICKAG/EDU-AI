# Web Based Training
## Overview

This web-based training is meant to introduce the learner into the topic of AI in image analysis for industrial automation use cases. There is a little story that guides the learner through the course.  
### It makes clear 
- why automating visual inspection is challanging, 
- why AI can help with the task
- how to differentiate the terms Artificial Intelligence, Machine Learning and Deep Learning
- what dataset is needed for a successful training 
- what a neural network is and how it works on a basic level  


### Target audience: 
learners with no prior experience in the field

### Training duration:
about 40 minutes
