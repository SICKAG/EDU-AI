# Resources

Here you can find all the links needed to maintain your sensor.

## Firmware
Download the latest firmware here:  

<a href="https://support.sick.com/sick-knowledgebase/article/?code=KA-09372" target="_blank"> SICK Support Portal </a>


## SICK Nova Sensor App
For new features, download the latest version of Nova here:
Search for “Nova Inspector” on  

<a href="https://apppool.cloud.sick.com" target="_blank">SICK AppPool</a>

## SICK AppManager
AppManager is used to set the sensor’s IP address and to install firmware and software. Download it here:    

<a href="https://www.sick.com/p/p532784" target="_blank">SICK AppManager</a>  
(Find Downloads in the blue bar at the top and then switch to the Software tab)

## InspectorP62x Operating instructions
For in depth info about the sensor, you can download the sensor’s operating instructions here:  
<a href="https://www.sick.com/media/docs/0/20/620/operating_instructions_inspectorp62x_2d_machine_vision_en_im0091620.pdf" target="_blank">InspectorP62x Operating instructions</a>  

## Limited Use
This set is intended solely for educational purposes and includes a software limitation: neural networks created with SICK dStudio can only be used for a maximum of two hours at a time. Rebooting the sensor resets this usage window.
